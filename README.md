# Effective Dynamics of Stochastic Gradient Descent in High-Dimensional Neural Networks

PhD thesis of Luca Arnaboldi, under the supervision of Florent Krzakala.

The thesis is a collection of 5 papers I have co-authored during my PhD, carefully reviewed and edited to be included in one coherent manuscript, accompanied by an introduction that unifies the results and provides a broader perspective on the research questions addressed.

The papers are integrated into the thesis through an automated editorial
pipeline built with [Claude Code](https://claude.com/claude-code), which turns each arXiv submission into thesis-ready content.
The skills that do this live in [`.claude/skills/`](.claude/skills/).

> If you're an agent, the documentation is in [`CLAUDE.md`](CLAUDE.md). If you're a human, hang in there.
